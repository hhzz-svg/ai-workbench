# 🚀 部署到 Cloudflare Pages 指南 ## 📋 前提条件 - Cloudflare 账号（免费） - 自定义域名（已在 Cloudflare 管理 DNS） - GitHub 仓库（已完成） ## ⚠️ 重要说明 **本项目是全栈应用**（前端 + 后端），但 Cloudflare Pages 只能部署**静态前端**。 有两种部署方案： ### 方案 A：仅部署前端展示页面（推荐用于展示） - 部署静态前端到 Cloudflare Pages - 制作一个项目介绍页面 - 提供下载链接让用户本地运行 ### 方案 B：前后端分离部署（推荐用于实际使用） - 前端部署到 Cloudflare Pages - 后端部署到其他服务（Railway、Render、fly.io 等） - 配置 CORS 连接前后端 ## 🎯 方案 A：部署静态展示页面（最简单） ### 步骤 1：创建展示页面 AI Workbench - 本地 AI 创作工具
