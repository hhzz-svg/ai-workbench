# 快速启动指南

## 启动服务

在项目根目录运行：

```powershell
.\start.ps1
```

或者局域网访问模式：

```powershell
.\start-lan.ps1
```

## 访问地址

- 本地访问：http://127.0.0.1:5173/
- 局域网访问：http://你的IP地址:5173/

## 首次配置

普通用户不需要安装 Codex CLI。首次打开工作台后：

1. 进入「设置」
2. 保存一个 OpenAI-compatible API 配置
3. 点击「测试连接」
4. 回到「新建」创建任务

如果你的电脑已经安装 Codex CLI，也可以在创建任务时选择「本机 Codex」作为可选运行环境。

## 部署到生产环境

### 1. 前端构建
```bash
cd frontend
npm run build
```

### 2. 静态文件
前端构建产物在 `frontend/dist/` 目录

### 3. 服务器配置
- 前端：配置 Nginx 或直接使用 Vite preview
- 后端：运行 FastAPI 服务（uvicorn）
- 反向代理：将 `/api` 转发到后端

### 4. 环境变量
确保设置：
- 前端：`VITE_API_BASE=https://你的后端地址`
- 后端：`SKILL_WORKBENCH_CORS_ORIGINS=https://你的前端域名`
- 输出目录权限
- 上传文件大小限制
