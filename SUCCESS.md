## 🚀 快速开始

```bash
# 1. 克隆仓库
git clone https://github.com/hhzz-svg/ai-workbench.git
cd ai-workbench

# 2. 安装依赖
python -m pip install -r backend/requirements.txt
npm --prefix frontend install

# 3. 启动服务
.\start.ps1

# 4. 访问
http://127.0.0.1:5173
```
