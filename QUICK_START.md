# 快速启动指南

## 启动服务

在项目根目录运行：

```powershell
.\start.ps1
```

或者局域网访问模式：

```powershell
.\start-lan.ps1
```

## 访问地址

- 本地访问：http://127.0.0.1:5173/
- 局域网访问：http://你的IP地址:5173/

## 部署到生产环境

### 1. 前端构建
```bash
cd frontend
npm run build
```

### 2. 静态文件
前端构建产物在 `frontend/dist/` 目录

### 3. 服务器配置
- 前端：配置 Nginx 或直接使用 Vite preview
- 后端：运行 FastAPI 服务（uvicorn）
- 反向代理：将 `/api` 转发到后端

### 4. 环境变量
确保设置：
- API 地址
- 输出目录权限
- 上传文件大小限制
